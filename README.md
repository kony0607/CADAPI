# CADAPI Scripts

このリポジトリは、CAD用のスクリプトのみをまとめたものです。
ベンダー配布のSDK/ヘッダ/ライブラリなどは含めていません。

## フォルダ一覧
- API/Scripts/L字フランジ
- API/Scripts/アルミ板固定部品
- API/Scripts/ギア作成
- API/Scripts/ピニオン作成入力画面
- API/Scripts/ラック作成
- API/Scripts/円弧ラックみぞ形

各フォルダに `.manifest` と `.py` が入っています。

## 使い方（概要）
1. 使いたいスクリプトのフォルダを、CADアプリのスクリプト配置場所にコピーします。
2. アプリのスクリプト/アドイン管理画面から読み込み・実行します。
3. 詳細は各スクリプトの `.manifest` を参照してください。

## 注意
- 配布元のSDKやAPI定義ファイルを同梱していません。必要な場合は公式配布物から入手してください。
